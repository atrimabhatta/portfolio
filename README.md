# Atrima Bhattacharyya — Portfolio

A cinematic developer portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

Designed with a modern dark aesthetic, immersive animations, responsive layouts, and interactive sections to showcase projects, skills, experience, and software engineering principles.

---

# ✨ Features

- Modern cinematic UI/UX
- Fully responsive design
- Animated hero section
- Interactive tech stack
- Principles & services sections
- Smooth scroll navigation
- Framer Motion animations
- Optimized Next.js architecture
- Custom dark theme
- Mobile-first layout

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

## Navigate to the project

```bash
cd YOUR_REPOSITORY
```

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
